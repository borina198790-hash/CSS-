Форма авторизации
Простая адаптивная форма входа с использованием современных CSS-техник

Описание проекта
Проект представляет собой базовую форму авторизации с использованием:
Flexbox-верстки
Адаптивного дизайна
Кастомных стилей для форм
Фонового изображения

Структура проекта
login-form/
├── images/
│   ├── background.png
│   ├── logo.png
│   ├── icon person_24px_outlined.svg
│   └── Password Off.png
├── index.html
├── index.css
└── README.md

Технические требования
Современный браузер с поддержкой CSS3
Подключение шрифта Raleway из Google Fonts
Корректные пути к изображениям в директории images/

Используемые технологии
HTML5 для структуры
CSS3 для стилизации
Flexbox для позиционирования
Google Fonts для типографики

Основные особенности
Адаптивный дизайн
Кастомные стили для форм
Анимации при наведении
Теневые эффекты
Центрирование элементов

Как использовать
Откройте index.html в браузере
Заполните поля формы
Нажмите кнопку “Войти”
